ESSARCH_TRANSFORMERS¶

Lists available transformers where the key is how we refer to it from a profile and the value is a class available in the PYTHONPATH:

ESSARCH_TRANSFORMERS = {
    'my_transformer': 'path.to.transformer.MyTransformer'
}
