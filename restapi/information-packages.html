GET /information-packages/¶

The information packages visible to the logged in user

Query Parameters:
 
  • agents ([string]) – agent ids
  • archivist_organization ([string]) – archivist organization names
  • state ([string]) – IP states
  • label ([string]) – labels
  • object_identifier_value ([string]) – object identifier values
  • responsible ([string]) – usernames
  • create_date_0 (date) – the earliest allowed create date
  • create_date_1 (date) – the latest allowed create date
  • entry_date_0 (date) – the earliest allowed entry date
  • entry_date_1 (date) – the latest allowed entry date
  • object_size_0 (int) – the lowest allowed IP size (bytes)
  • object_size_1 (int) – the highest allowed IP size (bytes)
  • start_date_0 (date) – the earliest allowed start date
  • start_date_1 (date) – the latest allowed start date
  • end_date_0 (date) – the earliest allowed end date
  • end_date_1 (date) – the latest allowed end date
